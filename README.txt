# Recomendador de Películas

## Descripción
Una app web sencilla que permite agregar películas y recomendar una aleatoriamente. Incluye una lista predefinida de películas que no se muestra en la interfaz.

## Instalación
1. Descarga los archivos `app.py` y `app_peliculas_con_lista_oculta.html`.
2. Instala Python y Flask: `pip install flask`.
3. Asegúrate de que ambos archivos estén en el mismo directorio.
4. Ejecuta la app: `python app.py`.

## Uso
- Abre `http://127.0.0.1:5000/` en tu navegador.
- Agrega películas usando el formulario.
- Haz clic en "Recomendar una Película" para obtener una recomendación.
